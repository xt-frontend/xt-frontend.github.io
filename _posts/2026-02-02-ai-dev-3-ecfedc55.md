AI 도구의 맥락 잃음 문제를 해결하는 CLAUDE.md  
AI 코드 도우미는 강력하지만, 프로젝트의 맥락을 알지 못해 반복적인 오류를 일으킨다. 예를 들어, npm 대신 pnpm을 사용하거나 TypeScript 프로젝트에 JavaScript 코드를 생성하는 등 개발자의 시간을 허비하게 만든다. 이 문제를 해결하기 위해 BuildrLab은 모든 프로젝트 루트에 `CLAUDE.md` 파일을 생성해 AI에게 필요한 정보를 집중적으로 전달한다.

`CLAUDE.md`는 AI가 프로젝트를 이해하는 핵심 문서다. 구조는 다음과 같다:  
1. **프로젝트 개요**: 2~3문장으로 목적과 대상자를 설명한다.  
2. **기술 스택**: 사용 중인 언어, 패키지 관리 도구(Vite, Vitest 등)를 명시한다.  
3. **폴더 구조**: AI가 파일을 생성할 위치를 정확히 알려준다.  
4. **공통 패턴**: 리팩토링 시 주의할 의존성이나 레거시 API 패턴을 기록한다.  

이 파일은 Claude Code 세션이 시작될 때 자동 로드되며, AI가 실수를 줄이고 개발자의 습관을 반영한 코드를 생성하도록 돕는다. 단순한 문서가 아닌, AI와 협업하는 개발자의 생산성 향상 전략이다. 프로젝트 맥락을 명확히 전달함으로써 AI가 도구가 아닌 팀원으로 기능하게 만드는 방법이다.
